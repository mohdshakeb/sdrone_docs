## Specification

### Size Mapping

| Size | Height | Padding (H) | Font Size |
|------|--------|-------------|-----------|
| `sm` | 32px | 12px | 12px |
| `md` | 40px | 16px | 14px |
| `lg` | 48px | 20px | 16px |

### Color Tokens

| State | Background | Border | Text |
|-------|------------|--------|------|
| Default | `--input-bg` | `--input-border` | `--fg-default` |
| Hover | `--input-bg` | `--input-border-hover` | `--fg-default` |
| Focus | `--input-bg` | `--input-border-focus` | `--fg-default` |
| Error | `--input-bg` | `--bg-negative-solid` | `--fg-default` |
| Disabled | `--bg-emphasis` | — | `--fg-muted` |

## Usage

<CodeBlock language="tsx" filename="example.tsx">
```tsx
import TextInput from '@/components/ui/TextInput';

// Basic usage
<TextInput
  placeholder="Enter your name"
  value={name}
  onChange={(e) => setName(e.target.value)}
/>

// With error state
<TextInput
  placeholder="Email address"
  hasError={!!errors.email}
  value={email}
  onChange={(e) => setEmail(e.target.value)}
/>

// Small size
<TextInput
  size="sm"
  placeholder="Search..."
/>
```
