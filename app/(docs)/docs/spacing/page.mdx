## Spacing Scale

We use a **proportional 4px scale**. The token number roughly represents the multiplier of 4px (with some exceptions like 0.5 for 2px).

Usage: `gap: var(--space-4)`, `padding: var(--space-8)`.

## Layout System

### Containers

#### Web Dashboard (Full Width)
For dashboard views with sticky sidebars.
- **Sidebar**: `240px` (`--sidebar-width`)
- **Content**: Fluid width

#### Web Dashboard Central (Fixed Width)
For centered content pages.
- **Max Width**: `800px`
- **Centered** in the available content area.

</Section>

## Usage

<CodeBlock language="tsx" filename="example.tsx">
```tsx
// Use spacing tokens for consistent spacing
<div style={{ gap: 'var(--space-4)', padding: 'var(--space-6)' }}>
  Content with 16px gap and 24px padding
</div>

// Layout containers
<div style={{ maxWidth: 'var(--content-max-width-central)' }}>
  Centered content (800px max)
</div>
```
