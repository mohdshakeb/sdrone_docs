# Icons

We use **Remix Icons** as our icon system. Currently, a curated set of icons is available via the `<Icon />` component.

<Section divider>

## Sizes

Standard icon sizes are **16px**, **20px** (Default), and **32px**.

## Usage

<CodeBlock language="tsx" filename="example.tsx">
```tsx
import { Icon } from '@/components/ui/Icon';

// Basic usage
<Icon name="settings" size={20} />

// With custom color
<Icon name="alert" color="var(--color-primary-600)" />

// With className
<Icon name="add" size={24} className="custom-class" />
```
