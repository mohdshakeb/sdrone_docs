# Navigation

Navigation components provide the structural framework for applications. They help users understand where they are and move between different sections.

<Section divider>

## Overview

The navigation system includes:
- **App Sidebar** (S-Drone prototype) - Fixed left sidebar with branding and navigation tabs
- **App Header** - Sticky header with page title, metadata, and actions

## Usage

<CodeBlock language="tsx" filename="layout.tsx">
```tsx
// In layout.tsx
import AppSidebar from '@/components/prototype/AppSidebar';
import AppHeader from '@/components/prototype/AppHeader';

  return (
    <div className="app-shell">
      <AppSidebar />
      <main className="main-content">
        <AppHeader />
        {children}
      </main>
    </div>
  );
}
```
