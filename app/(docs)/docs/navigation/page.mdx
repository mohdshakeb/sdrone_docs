# Navigation

Navigation components provide the structural framework for applications. They help users understand where they are and move between different sections.

<Section divider>

## Overview

The navigation system includes:
- **App Sidebar** (S-Drone prototype) - Fixed left sidebar with branding and navigation tabs
- **App Header** - Sticky header with 4 variants:
  - `default` â€” Page title, "Start New" dropdown, "Report Incident" button, and SOS
  - `form` â€” Back button, form title, step progress, cancel/submit actions
  - `innerPage` â€” Back button, title, breadcrumbs, optional export action
  - `docs` â€” Design System branding with theme toggle and prototype link
