# Radio Group

The RadioGroup component allows selection of a single option from multiple choices with support for descriptions and layouts.

<Section divider>

## Variants

### Vertical Layout (Default)

<Preview>
  <RadioGroup
    name="example-vertical"
    options={[
      { value: 'option1', label: 'Option 1', description: 'This is the first option' },
      { value: 'option2', label: 'Option 2', description: 'This is the second option' },
      { value: 'option3', label: 'Option 3', description: 'This is the third option' },
    ]}
    aria-label="Example radio group"
  />
</Preview>

### Horizontal Layout

<Preview>
  <RadioGroup
    name="example-horizontal"
    options={[
      { value: 'yes', label: 'Yes' },
      { value: 'no', label: 'No' },
      { value: 'maybe', label: 'Maybe' },
    ]}
    direction="horizontal"
    aria-label="Horizontal radio group"
  />
</Preview>

## Usage

<CodeBlock language="tsx" filename="example.tsx">
```tsx
import RadioGroup from '@/components/ui/RadioGroup';

const options = [
  { value: 'low', label: 'Low Priority' },
  { value: 'medium', label: 'Medium Priority' },
  { value: 'high', label: 'High Priority', description: 'Requires immediate attention' },
];

// Basic usage
<RadioGroup
  name="priority"
  options={options}
  value={priority}
  onChange={setPriority}
  aria-label="Priority level"
/>

// Horizontal layout
<RadioGroup
  name="consent"
  options={[
    { value: 'yes', label: 'I agree' },
    { value: 'no', label: 'I decline' },
  ]}
  direction="horizontal"
  value={consent}
  onChange={setConsent}
/>
```
