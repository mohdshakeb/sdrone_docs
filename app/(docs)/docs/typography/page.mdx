# Typography

We use **Inter** as our primary typeface. It is a variable font with a tall x-height, making it highly legible at small sizes.

## Font Family

</div>

## Tokens

Underlying CSS variables used to construct the styles above.

### Weights
- **Regular (400)**: `var(--font-weight-regular)`
- **Medium (500)**: `var(--font-weight-medium)`
- **Medium (600)**: `var(--font-weight-semibold)`

### Sizes
- **10px**: `var(--text-10)`
- **12px**: `var(--text-12)`
- **14px**: `var(--text-14)`
- **16px**: `var(--text-16)`
- **24px**: `var(--text-24)`
