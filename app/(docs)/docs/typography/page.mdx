# Typography

We use **Inter** as our primary typeface. It is a variable font with a tall x-height, making it highly legible at small sizes.

## Usage

<CodeBlock language="tsx" filename="example.tsx">
```tsx
// Use utility classes for consistent typography
<h1 className="text-heading">Page Title</h1>
<p className="text-body">Body text content</p>
<span className="text-caption">Caption text</span>

// Or use CSS variables directly
<div style={{ fontSize: 'var(--text-14)', fontWeight: 'var(--font-weight-medium)' }}>
  Custom text
</div>
```
