# Buttons

Buttons allow users to take actions and make choices with a single tap. They are the primary interactive element for triggering events.

---

## Overview

The `Button` component supports multiple variants and sizes. All buttons use a **pill shape** (fully rounded corners).

## Properties

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | — | Button label (required) |
| `size` | `string` | `"md"` | `xs`, `sm`, `md`, `lg` |
| `variant` | `string` | `"primary"` | `primary`, `secondary`, `negative`, `notice`, `positive` |
| `leadingIcon` | `ReactNode` | — | Icon before label |
| `trailingIcon` | `ReactNode` | — | Icon after label |
| `disabled` | `boolean` | `false` | Disables the button |
| `href` | `string` | — | Renders as a link |

---

## Specification

### Variants

| Variant | Token / Color | Purpose |
|---------|---------------|---------|
| `primary` | `--bg-accent` | Main call to action |
| `secondary` | `--bg-strong` | Neutral supporting actions |
| `negative` | `--bg-negative-intense` | Destructive actions (Logout, Delete) |
| `notice` | `--bg-notice-intense` | Warning actions |
| `positive` | `--bg-positive-intense` | Success/Confirmation actions |

### Size Mapping

| Size | Height | Padding (H) | Font Size | Recommended Icon |
|------|--------|-------------|-----------|------------------|
| `xs` | 28px | 12px | 12px | 16px |
| `sm` | 32px | 16px | 12px | 16px |
| `md` | 40px | 24px | 14px | 20px |
| `lg` | 48px | 32px | 16px | 24px |

---

## Usage

```tsx
import Button from '@/components/ui/Button';
import { Icon } from '@/components/ui/Icon';

// Variants
<Button variant="primary">Primary</Button>
<Button variant="secondary">Secondary</Button>

// Status Variants
<Button variant="negative">Delete</Button>
<Button variant="positive">Confirm</Button>

// Icon and Size
<Button variant="secondary" size="lg" leadingIcon={<Icon name="add" size={20} />}>
  Create New
</Button>
```
